Demo — Widget Divina

Questa pagina simula un sito FORMA o ATS. In basso a destra trovi la bolla di chat 💬. Il widget chiama POST /chat del servizio Divina usando la chiave del tenant.

Pronto all'uso: il widget punta al servizio live in cloud (divina.formahub.it). Apri questo file nel browser e prova la chat. Per testare lo scope, cambia data-key qui sotto in CHIAVE_ATS e verifica che non risponda sulle domande interne FORMA.

Novità v2: premi 🎤 per parlare (dettatura vocale) e l'icona 🔊 nell'header per farti leggere le risposte. Il widget gira in Shadow DOM: non eredita né rompe i CSS del sito ospite. La voce usa le API del browser (gratis) — funziona meglio su Chrome/Edge.
